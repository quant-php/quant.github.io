Equatable

Reference

Definition​

Namespace: Quant\Core\Contract

Equatable implements an equivalence relation for objects.

interface Equatable

Example​

#[Getter]
class Point implements Equatable {

    use AccessorTrait;
    
    public function __construct(
        private float $x,
        private float $y
    ) {
    } 

    public function equals(Equatable $b): array
    {
        return ($b instanceof Point) && $b->getX() === $this->getX() && $b->getY() === $this->getY();
    }

}

Remarks​

This interface implements an equivalence relation for objects of T, with T being a subtype of Comparable. It provides a method equals whose implementation must, for all a,b,c∈Ta, b, c \in Ta,b,c∈T, conform to (a∈T) ∼ (b∈T):  ⟺  a(a \in T) \space \thicksim \space (b \in T) :\iff a(a∈T) ∼ (b∈T):⟺a->equals(b)equals(b)equals(b), that is
